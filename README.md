# Evaluación Sumativa 4

Para esta evaluación deberán crear la API pedida con el uso de DJango
Dicha API debe proporcionar de datos de productos a este Frontend


No se debe preocupar por el frontend ya que está hecho para que consuma los datos que usted le pasará mediante el uso de los siguientes endpoints:
1. localhost:8000/api/producto/ -> Este trae un producto de manera aleatoria
2. localhost:8000/api/productos/ -> Es en plural e indica que trae a todos los productos pero este requiere de un token
3. localhost:8000/api/token/ -> Token de autenticación, devuelve un JWT en caso de ser válido
